Características da Linguagem Python

Python é uma linguagem interpretada!

O interpretador é um programa que aceita comandos escritos em Python e os executa linha por linha.

É ele quem vai traduzir nossos programas em um formato que possa ser executado pelo computador.

Sem um interpretador, nossos programas são apenas textos.

Cuidados ao digitar

  1. Letras maiúsculas e minúsculas são diferentes

  2. Aspas não devem ser esquecidas para indicar textos
  3. Parênteses não são opcionais
  4. Espaços são importantes

Algoritmos

Conjunto de instruções descrita de forma clara e precisa.

Não pode ser ambíguo! Cada instrução deve ser precisa, sem permitir mais de uma interpretação.

Algoritmos são capazes de realizar:

Ler e escrever dados.

Avaliar expressões algébricas, relacionais e lógicas.

Tomar decisões com base nos resultados das expressões avaliadas.

Repetir um conjunto de ações de acordo com uma condição.

Algoritmos
